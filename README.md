# 🛡️ Spambot Detection Using Interpretable AI

## 📖 About the Project

This project is a Machine Learning-based web application that detects fake follower (spambot) accounts on social media. Unlike traditional models, it uses **Interpretable AI** to explain why an account is classified as a spambot, making the predictions more transparent and trustworthy.

## ✨ Features

* Detects fake follower (spambot) accounts.
* Provides explanations for each prediction using Interpretable AI.
* Simple web interface built with Django.
* Machine Learning-based classification.

## 🛠️ Technologies Used

* Python
* Django
* Scikit-learn
* Pandas
* NumPy
* HTML & CSS

## 📂 Project Structure

* manage.py – Runs the Django application.
* Remote_User – User-side application logic.
* Service_Provider – Admin and backend modules.
* Template – HTML templates.
* Datasets – Dataset used for model training.
